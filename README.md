# Pyrenees

An interactive 3D map using Mapbox. Shows the GR10 and GR11 trails, and allows for quick switching between Outdoor and Satellite maps.

## Current and future functionality

- [x] 3D terrain
- [x] GR10 and GR11 trails
- [x] Quick switching between Outdoor and Satellite maps
  - [ ] Preview in switcher
- [ ] Shelters
- [ ] Route planning
  - [x] Route drawing and clearing
  - [x] Undo/redo
  - [ ] Route editing
  - [ ] Route info
  - [x] Local storage
  - [ ] Export/import
  - [ ] Route waypoints (shelters)
  - [ ] Route splitting
  - [ ] Multiple routes
- [ ] Improve memory usage?
